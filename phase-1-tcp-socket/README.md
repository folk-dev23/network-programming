# Phase 0 - Internet Basics

## สิ่งที่เรียนรู้
- IP Address (Private vs Public)
- Port คืออะไร และ Port สำคัญที่ต้องจำ
- DNS ทำงานยังไง

## Commands ที่ใช้
- ipconfig → ดู Private IP
- curl ifconfig.me → ดู Public IP
- netstat -an | grep LISTENING → ดู Port ที่เปิดอยู่
- nslookup → แปลง domain เป็น IP
